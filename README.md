## 실행 및 URL

### backend
```sh
$ cd back-end
$ gradlew bootJar
$ gradlew bootRun
```

### frontend
```sh
$ cd front-end
$ npm run serve
```

### Access server side using following URL

```
http://localhost:5000
```

### Access application using following URL

```
http://localhost:3000
```
